# How To

Karena udah sesuai sama tree yang ada bisa di skip :D

```bash
chmod +x run
./run
```

Kerjain ini aja

```bash
cd modelLangProcInJava
chmod +x compile
./compile
```
Untuk menajalankan `testcase`

```bash
./testcase <nomor-tc>
```
